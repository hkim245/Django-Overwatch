Create Hero Assignments Tutorial



1. Select number of players

2. Fill out all player fields

a) Go to a player's profile page on overbuff.com and copy the

second to last "/" character in the web url to the end of the web url

b) Paste the copied text into the player field

3. Select at least the number of heroes that is equal to or greater

than the number of player fields

4. Press the Create Hero Assignments button

5. Receive the most optimal hero assignments with each

player paired with the most optimal hero
